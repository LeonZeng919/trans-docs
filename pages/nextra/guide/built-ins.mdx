# 内置组件

Nextra 包含了一些内置组件，您可以使用它们来更好地样式化您的内容：

<Cards>
  <Card
    icon={<WarningIcon />}
    title="Callout"
    href="/nextra/guide/built-ins/callout"
  />
  <Card icon={<CardsIcon />} title="Tabs" href="/nextra/guide/built-ins/tabs" />
  <Card
    icon={<IdCardIcon />}
    title="Cards"
    href="/nextra/guide/built-ins/cards"
  />
  <Card icon={<OneIcon />} title="Steps" href="/nextra/guide/built-ins/steps" />
  <Card
    icon={<FolderTreeIcon className="h-5 w-5" />}
    title="FileTree"
    href="/nextra/guide/built-ins/filetree"
  />
</Cards>
