# 文档主题

Nextra 文档主题是一个包括几乎您构建现代文档网站所需一切的主题。它包含顶部导航栏、搜索栏、页面侧边栏、目录侧边栏以及其他内置组件。

这个网站本身就是使用 Nextra 文档主题构建的。

## 从模板快速开始

### 部署到 Vercel

您可以通过点击以下链接开始创建您自己的 Nextra 站点并部署到 Vercel：

<a
  className="mt-3 inline-flex"
  target="_blank"
  href="https://vercel.com/new/clone?s=https%3A%2F%2Fgithub.com%2Fshuding%2Fnextra-docs-template&showOptionalTeamCreation=false"
>
  ![](https://vercel.com/button)
</a>

Vercel 将 fork [Nextra 文档模板](https://github.com/shuding/nextra-docs-template) 并为您部署网站。一旦完成，仓库中的每次提交都将自动部署。

### Fork 模板

您也可以手动 fork [模板仓库](https://github.com/shuding/nextra-docs-template).

## 作为新项目开始

<Steps>
### 安装

要手动创建一个 Nextra 文档站点，您需要安装 **Next.js**、**React**、**Nextra** 和 **Nextra 文档主题**。在您的项目目录中，运行以下命令来安装依赖项：

```sh npm2yarn
npm i next react react-dom nextra nextra-theme-docs
```

<Callout>
  如果您的项目中已经安装了 Next.js，您只需要安装 `nextra` 和 `nextra-theme-docs`
  作为附加组件。
</Callout>

在 `package.json` 中添加以下脚本：

```json filename="package.json"
"scripts": {
  "dev": "next",
  "build": "next build",
  "start": "next start"
},
```

您可以启动服务器，例如如果您使用 npm，运行 `npm run dev`，这将在开发模式下运行或 `npm run build && npm run start` 在生产模式下运行。

<Callout>
  如果您不熟悉 Next.js，请注意开发模式显著慢，因为 Next.js
  会编译您导航到的每个页面。
</Callout>

### 添加 Next.js 配置

在项目的根目录中创建以下 `next.config.js` 文件：

```js filename="next.config.js"
const withNextra = require("nextra")({
  theme: "nextra-theme-docs",
  themeConfig: "./theme.config.jsx",
});

module.exports = withNextra();

// 如果您有其他 Next.js 配置，可以将它们作为参数传递：
// module.exports = withNextra({ /* other next.js config */ })
```

通过上述配置，Nextra 可以在您的 Next.js 项目中处理 Markdown 文件，并使用指定的主题。其他 Nextra 配置可以在[指南](/nextra/guide)中找到。

### 创建文档主题配置

最后，在项目的根目录中创建相应的 `theme.config.jsx` 文件。这将用于配置 Nextra 文档主题：

```jsx filename="theme.config.jsx"
export default {
  logo: <span>My Nextra Documentation</span>,
  project: {
    link: "https://github.com/shuding/nextra",
  },
  // ... other theme options
};
```

完整的主题配置可以在[这里](/nextra/docs-theme/theme-configuration)找到。

### 准备就绪！

现在，您可以创建您的第一个 MDX 页面作为 `pages/index.mdx`：

```mdx filename="pages/index.mdx"
# 欢迎使用 Nextra

Hello, world!
```

然后运行 `package.json` 中指定的 `next` 或 `next dev` 命令来开始开发项目！🎉

</Steps>
