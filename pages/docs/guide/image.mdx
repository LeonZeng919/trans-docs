# Next.js 图片

在MDX中使用[Next.js 图片](https://nextjs.org/docs/basic-features/image-optimization)的标准方法是直接导入组件：

```mdx
import Image from "next/image";

<Image src="/demo.png" alt="Hello" width={500} height={500} />
```

## 静态图片

import { Callout } from "nextra/components";

<Callout>此功能默认通过在Nextra配置中设置`staticImage: true`来启用。</Callout>

Nextra支持使用Markdown语法自动优化您的静态图片导入。您不需要指定图片的宽度和高度，只需使用`![]()` Markdown语法：

```md filename="index.mdx"
![Hello](/demo.png)
```

这将加载`public`文件夹内的`demo.png`文件，并自动将其封装在Next.js的`<Image>`中。

<Callout>
  如果您不想通过`public`托管图片，也可以使用`![](../public/demo.png)`从相对路径加载图片。
</Callout>

使用Next.js 图片，将不会有布局移动，并且在加载图片时，默认会显示一个美观的模糊占位符：
