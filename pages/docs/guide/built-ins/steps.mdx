# Steps 组件

一个内置组件，用于将编号列表转换为步骤的视觉表示。

## 示例

<Steps>

### 这是第一步

这是第一步的描述。

### 这是第二步

这是第二步的描述。

### 这是第三步

这是第三步的描述。

</Steps>

## 使用方法

用 `Steps` 组件包裹一组 markdown h3 标题，将它们转换为视觉步骤。

<Steps>

### 步骤 1

步骤 1 的内容。

### 步骤 2

步骤 2 的内容。

</Steps>
