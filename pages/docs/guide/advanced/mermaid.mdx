# Mermaid

Nextra 支持 [mermaid](https://mermaid.js.org) 图表。就像在 GitHub 中一样，您可以通过使用 `mermaid` 代码块语言在 markdown 文件中使用它。开箱即用，Nextra 使用
[`@theguild/remark-mermaid`](https://npmjs.com/package/@theguild/remark-mermaid)
包来替换代码块为 `<Mermaid />` 组件。

## 示例

```mermaid
graph TD;
subgraph AA [消费者]
A[移动应用];
B[网页应用];
C[Node.js 客户端];
end
subgraph BB [服务]
E[REST API];
F[GraphQL API];
G[SOAP API];
end
Z[GraphQL API];
A --> Z;
B --> Z;
C --> Z;
Z --> E;
Z --> F;
Z --> G;
```
