# Npm2Yarn

Nextra 使用
[`@theguild/remark-npm2yarn`](https://npmjs.com/package/@theguild/remark-npm2yarn)
包替换含有 `npm2yarn` 元数据的代码块，
使用来自 `nextra/components` 的
[`<Tabs />` 和 `<Tabs.Tab />` 组件](/docs/guide/built-ins/tabs)。

选定的标签页会保存在本地存储中，未来页面渲染时将选择该标签页。

## 示例

```sh npm2yarn
npm i -D @graphql-eslint/eslint-plugin
```
